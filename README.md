# Customer Churn Analysis & Prediction System

## Objective
To analyze customer behavior and identify factors contributing to churn, enabling data-driven retention strategies.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Key Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Churn prediction using Logistic Regression
- Interactive dashboard

## Business Insights
- Month-to-month customers show higher churn
- High monthly charges increase churn risk
- New customers are more likely to churn

## Conclusion
This project demonstrates how analytics and machine learning can help organizations reduce customer attrition.
